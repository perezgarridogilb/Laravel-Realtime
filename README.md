### Tiempo real y Laravel: Múltiples apps realtime con Laravel

Este repositorio implementa una arquitectura robusta para aplicaciones web y móviles en tiempo real utilizando **Laravel Echo** y **WebSockets** independientes (compatibles con el protocolo Pusher). Demuestra la gestión de canales públicos y privados

## ⚡ Características Principales

* **Conexión WebSockets Bidireccional:** Comunicación full-duplex servidor-cliente instantánea.
* **Broadcasting Asíncrono:** Uso de colas de Laravel (`ShouldBroadcast`) para no bloquear el hilo principal.
* **Canales Privados:** Implementación de autenticación y seguridad en la escucha de eventos sensibles.

## 🛠️ Pila Tecnológica

* **Backend:** Laravel Framework (Events & Listeners)
* **WebSockets Server:** Pusher Protocol
* **Frontend:** JavaScript & Laravel Echo
